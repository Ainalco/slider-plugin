Converts English dates like 2025-01-20 into beautiful Bangla date formats. Simple, lightweight, and automatic.

== Description ==

This plugin automatically detects dates in your content and converts them into Bangla (বাংলা) format. Supports multiple formats, a settings toggle, and a shortcode to show today’s date in Bangla.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/bangla-date-converter`.
2. Activate through the WordPress admin.
3. Go to Settings > Bangla Date Converter to configure.
